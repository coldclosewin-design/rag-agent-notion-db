# 벡터 스토어와 임베딩

## 임베딩(Embedding)

임베딩은 텍스트를 고정 길이의 숫자 벡터로 변환한 것이다.
의미가 비슷한 문장은 벡터 공간에서 가까운 위치에 놓이도록 학습되어 있어,
"의미 기반 검색(semantic search)"의 기반이 된다.

이 프로젝트는 OpenAI의 `text-embedding-3-small` 모델을 사용한다.
1536차원 벡터를 생성하며 한국어를 포함한 다국어를 지원하고 비용이 저렴하다.

## 벡터 스토어(Vector Store)

벡터 스토어는 임베딩 벡터를 저장하고, 주어진 쿼리 벡터와 가장 유사한 벡터를
빠르게 찾아주는(최근접 이웃 탐색) 데이터베이스다.

### Chroma

Chroma는 로컬에서 파일 기반으로 동작하는 오픈소스 벡터 스토어다.
별도 서버 없이 파이썬 라이브러리만으로 사용할 수 있어 스터디/프로토타입에 적합하다.
`persist_directory`를 지정하면 디스크에 저장되어 프로세스를 재시작해도 색인이 유지된다.

## 유사도 검색

코사인 유사도(cosine similarity)가 가장 일반적이다.
두 벡터 사이의 각도가 작을수록(코사인 값이 1에 가까울수록) 의미가 유사하다고 본다.
검색 시 top-k(예: 4개)의 가장 유사한 청크를 가져와 LLM의 컨텍스트로 사용한다.
